# Review History - 22-04-2026

## Summary
Fix Thymeleaf template crash and implement product review management system for admin.

## Issues Fixed

### 1. IndexOutOfBoundsException in Templates
**Error:** `product.images.get(0).imageUrl` threw exception when product has empty images list.

**Solution:** Added null check in Thymeleaf:
```html
th:value="${not #lists.isEmpty(product.images) ? product.images[0].imageUrl : '/images/no-image.jpg'}"
```

**Files:**
- `templates/index.html`
- `templates/user/product-list.html`
- `templates/user/product-detail.html`

## New Features

### 2. Product Review Management

#### Backend
- Added `isVisible` field to `ProductReview` (default: false, requires admin approval)
- New repository methods: `findByIsVisibleFalse()`, `countByIsVisibleFalse()`
- New service methods:
  - `getPendingReviews()` - get all pending reviews
  - `getPendingReviewCount()` - get count of pending reviews
  - `approveReview(Long id)` - approve a review
  - `getAllReviewsForProduct(Long productId)` - get reviews for specific product
  - `getPendingReviewCountByProduct()` - get pending count per product
- New DTO: `ProductReviewCount` (productId, count)

#### Frontend
- Added review button with badge in product-manage.html
- New product-reviews.html page showing pending/approved reviews

#### Access
- `/admin/seafood-products` → click blue review button
- `/admin/seafood-products/{id}/reviews`
