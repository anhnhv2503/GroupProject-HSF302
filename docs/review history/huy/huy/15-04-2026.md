## Phiên 1 — Thiết lập ứng dụng và cấu hình Database (SQL Server)

**Thời gian:** 08:30 — 10:30  
**Công cụ AI:** Antigravity / Gemini 

### Câu hỏi của tôi

> Bạn hãy hướng dẫn cách cấu hình file `application.properties` trong Spring Boot 3 để kết nối với cơ sở dữ liệu SQL Server cho dự án SeafoodStoreAI. Yêu cầu có cấu hình hiển thị câu lệnh SQL và tự động cập nhật schema (hibernate.ddl-auto).

### Câu trả lời của AI

```properties
# Cấu hình Database SQL Server
spring.datasource.url=jdbc:sqlserver://localhost:1433;databaseName=SeafoodStoreAI;encrypt=true;trustServerCertificate=true;
spring.datasource.username=sa
spring.datasource.password=your_password
spring.datasource.driver-className=com.microsoft.sqlserver.jdbc.SQLServerDriver

# Cấu hình JPA & Hibernate
spring.jpa.show-sql=true
spring.jpa.properties.hibernate.format_sql=true
spring.jpa.hibernate.ddl-auto=update
spring.jpa.database-platform=org.hibernate.dialect.SQLServer2012Dialect

# Ứng dụng
spring.application.name=SeafoodStoreAI
server.port=8080
```
AI cũng đã giải thích kỹ ý nghĩa của từng cấu hình, đặc biệt là `trustServerCertificate=true` rất quan trọng khi sử dụng môi trường local.

### Tôi đã làm gì với câu trả lời đó

- **Áp dụng trực tiếp:** Thay đổi thông tin password cho phù hợp với máy local, cấu hình thành công kết nối giữa Spring Boot và SQL Server.

## Phiên 2 — Tạo cấu trúc Homepage Controller & Định tuyến Thymeleaf

**Thời gian:** 10:45 — 12:00  
**Công cụ AI:** Antigravity / Gemini 

### Câu hỏi của tôi

> Làm thế nào để tạo một `HomeController` đơn giản trả về template Thymeleaf trang chủ `index.html` của cửa hàng?

### Câu trả lời của AI

```java
package com.group.seafoodstoreai.controller;

import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;

@Controller
public class HomeController {

    @GetMapping("/")
    public String home(Model model) {
        model.addAttribute("pageTitle", "Trang Chủ");
        // Có thể load các danh sách sản phẩm nổi bật (bestSellers) ở đây sau khi có logic database
        return "index"; 
    }
}
```
AI nhắc nhở cần tạo file `index.html` bên trong thư mục `src/main/resources/templates/` kèm theo việc khai báo namespace `xmlns:th="http://www.thymeleaf.org"` ở đầu file HTML.

### Tôi đã làm gì với câu trả lời đó

- **Chỉnh sửa & áp dụng:** Đưa code vào thư mục controller, tạo file `index.html` chứa layout. Chạy ứng dụng qua port 8080 và đã thấy view trang chủ hiển thị thành công mà không có lỗi mapping.
